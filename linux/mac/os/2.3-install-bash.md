### Install Bash on MacOS
The default shell on macOS is `zsh`, but you can install and use `bash` if you prefer it. Follow these steps to install `bash` using Homebrew:

### Step 1: Install Bash
```
brew install bash
```

### Step 2: Verify the Installation
```
bash --version
```

### Step 3: Change the Default Shell to Bash
```
chsh -s /opt/homebrew/bin/bash
```

### Step 4: Verfify the Default Shell Change
```
echo $SHELL
```
```
echo $0
```
### Step 5: Configure bash Profile
Create or edit the `.bash_profile` file in your home directory to customize your bash environment. An example can be found in [bash-profile.txt](bash_profile/bash-profile.txt).
